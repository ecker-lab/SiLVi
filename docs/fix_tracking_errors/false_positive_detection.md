# Fixing False Positive Detection

1. Click on the track.
1. Go to the tracking panel. 
1. Select the <span class="badge text-bg-light">Remove track</span> option. 
1. Under the <span class="badge text-bg-dark">Apply changes for</span> section.
    1. To remove all instances of the track, select <span class="badge text-bg-light">All frames from the beginning</span> 
    1. To remove the instances of the track only after the current frame, select <span class="badge text-bg-light">All frames after this frame</span>.
1. Click <span class="badge text-bg-primary">Save</span> to confirm.
