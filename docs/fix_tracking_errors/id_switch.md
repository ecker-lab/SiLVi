# Fixing ID Switch

1. Click on the track on the frame where the ID switch is happening.
1. Go to the tracking panel.
1. Select <span class="badge text-bg-light">Assign the next unused ID</span>
1. Under the <span class="badge text-bg-dark">Apply changes for</span> section, select <span class="badge text-bg-light">All frames after this frame</span>.
1. Click <span class="badge text-bg-primary">Save</span> to confirm.
